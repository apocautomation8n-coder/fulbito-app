# Politica de privacidad - borrador

Ultima actualizacion: 21 de mayo de 2026

Fulbito es una aplicacion desarrollada por Apoc Automation para conectar jugadores y clubes de futbol en Cordoba, Argentina.

Contacto de soporte: apoc@apocautomation.site

Sitio web: https://apoc.apocautomation.site

## Datos que recopilamos

- Datos de cuenta: nombre, email, rol de usuario y foto de perfil.
- Datos de jugador: edad o fecha de nacimiento, posicion, nivel de juego, historial de partidos y calificaciones.
- Datos de club: nombre del club, direccion, barrio, ubicacion, fotos, canchas, precios, disponibilidad y politicas de cancelacion.
- Datos de reservas: cancha, fecha, horario, estado de pago, monto y estado de cancelacion.
- Datos de pago: identificadores de pago y estado de transaccion procesados por MercadoPago.
- Datos de ubicacion: ubicacion aproximada o precisa para mostrar canchas cercanas, si el usuario otorga permiso.
- Datos de dispositivo: token de notificaciones push para enviar avisos de reservas, pagos y partidos.
- Reportes y moderacion: bloqueos, reportes y comentarios vinculados a usuarios o partidos.

## Para que usamos los datos

- Crear y proteger cuentas de usuario.
- Mostrar canchas cercanas y disponibilidad.
- Procesar reservas, pagos, senas, split de pagos y cancelaciones.
- Permitir que jugadores soliciten unirse a partidos.
- Verificar clubes y prevenir abusos.
- Enviar notificaciones relacionadas con reservas, pagos y partidos.
- Cumplir obligaciones legales, de seguridad y soporte.

## Servicios de terceros

Fulbito puede usar:

- Supabase para autenticacion, base de datos, almacenamiento y tiempo real.
- MercadoPago para pagos.
- Firebase Cloud Messaging para notificaciones push.
- OpenStreetMap o proveedores de mapas para visualizacion de ubicaciones.

## Pagos

Los pagos son procesados por MercadoPago. Fulbito no almacena numeros completos de tarjetas ni credenciales bancarias.

## Ubicacion

La ubicacion se usa para mostrar canchas cercanas. Si el usuario no otorga permiso, puede buscar por barrio.

## Eliminacion de cuenta

El usuario podra solicitar la eliminacion de su cuenta desde soporte o desde la app cuando esta funcionalidad este habilitada. Al eliminar una cuenta, Fulbito puede conservar registros minimos necesarios para soporte, seguridad, pagos, disputas o cumplimiento legal.

## Menores

Fulbito esta orientada a usuarios mayores de 18 anos.

## Cambios

Apoc Automation puede actualizar esta politica. Los cambios importantes se comunicaran dentro de la app o por canales de soporte.
